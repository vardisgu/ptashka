# Product

## Register

brand

## Platform

web

## Users

Гости заведений в Казани: молодая городская аудитория, приходящая из Instagram/сторис (в основном с телефона). Смотрят, что за место, выбирают, куда пойти сегодня — утром на завтрак, днём за фильтром, вечером на вино или мероприятие.

## Product Purpose

Хаб-страница трёх заведений одной команды: кофейня «Ранняя пташка» (Астрономическая, 17), батч-бар «Ясный» (Островского, 9), винный бар «Мелок» (Кави Наджми, 8а). По решению заказчика (июль 2026) хаб — типографская «вывеска» в духе masamadre.ru (утверждённый референс): тёплый белый фон #fcfafc, моноширинный шрифт строчными, три чёрных логотипа столбиком с mono-подписями (формат · чем известно · адрес), бегущая строка внизу, единственный акцент — красный на ховере. Без фотографий, иллюстраций и WebGL. Атмосферные фотографии живут на страницах заведений. Успех — переход в раздел/профиль заведения. Домен: ptashka.ru.

## Conversion & proof

- Primary CTA: клик по сегменту заведения → переход на его страницу (пока — Instagram-профиль).
- Secondary CTA: нет (v1 — только hero).
- Строка, которую посетитель запоминает: «одна команда — три разных места в Казани».
- Belief ladder: это три разных места → они мне по-разному нужны (утро/день/вечер) → выберу и пойду.
- Proof: живые фото заведений (public/cover), узнаваемые логотипы (public/logo).

## Brand Personality

Три голоса под одной крышей, у каждого сегмента своя айдентика, унаследованная от заведения:
- «Ранняя пташка» — тёплое утро, дерево, растения, солнце; коралловый логотип; фирменный жест — взлетающая птичка.
- «Ясный» — светлый, выбеленный, минималистичный; жёлтый рукописный логотип со звёздочкой; фирменный жест — сияние/вспышка.
- «Мелок» — вечерний, индустриальный, «бар с характером»; белый логотип; фирменный жест — разливающееся вино.
Общий тон: уверенный, немногословный, без маркетингового пафоса.

## Anti-references

- Шаблонные ресторанные сайты-визитки (слайдер + меню PDF + жёлтые кнопки брони).
- AI-лендинг: кремовый фон, eyebrow-подписи над каждой секцией, карточная сетка, градиентный текст.
- Единый «фирстиль группы», стирающий различия трёх мест, — различие и есть сообщение.

## Design Principles

- Фото и есть дизайн: полноэкранная фотография несёт всю атмосферу, хром страницы минимален.
- Три лица, один дом: сегменты обязаны выглядеть по-разному, но жить в одной композиции.
- Один жест на бренд: у каждого сегмента ровно одна фирменная hover-анимация, ничего лишнего.
- Мобильный — первый: аудитория приходит из Instagram; тап-механика равноценна hover, не урезана.

## Accessibility & Inclusion

prefers-reduced-motion обязателен (анимации-жесты отключаются, остаются fade). Контраст подписей поверх фото — через затемняющий scrim, WCAG AA. Hover-эффекты только под @media (hover: hover). Сегменты — настоящие ссылки, доступные с клавиатуры (focus-visible = состояние hover).
